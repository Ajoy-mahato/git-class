bydefault git ignore empty folder 
if you want to add empty folder then you can create a .gitkeep file in your empty folder

----------branch---------------
1.To create a new branch, you can use the following command:-
git branch
git branch bug-fix
git switch bug-fix
git log
git switch main
git switch -c dark-mode
git checkout orange-mode


-----------merging----
git checkout main
git merge bug-fix

--------merging conflict probel-----
error: Merging is not possible because you have unmerged files.
hint: Fix them up in the work tree, and then use 'git add/rm <file>'
hint: as appropriate to mark resolution and make a commit.
fatal: Exiting because of an unresolved conflict.

--------how to solve------
use "git merge --abort" to abort the merge
